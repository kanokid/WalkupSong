# Walkup Song

Walkup Song is a simple Android app that allows you to create a batting order for a softball team and assign a walk-up song to each player. The app uses the Deezer or Spotify API to search for songs and play 30-second snippets.

## Features

*   **Batting Order Management:** Add, remove, and reorder players in the batting list.
*   **Song Search:** Search for songs using the Deezer or Spotify API.
*   **Music Playback:** Play 30-second snippets of songs.
*   **Material 3 UI:** The app is built with the latest Material 3 design guidelines, providing a modern and expressive user interface.
*   **Data Persistence:** The batting order is saved to your device, so you don't have to re-enter it every time you open the app.
*   **API Selection:** Choose between Deezer and Spotify as your music source.

## Getting Started

To get started with Walkup Song, you'll need to have Android Studio installed.

1.  **Clone the repository.**
2.  **Open the project in Android Studio.**
3.  **Build and run the app.**

## How to Use

1.  **Add players to the batting order:** Enter a player's name and number and tap the "Add Player" button.
2.  **Select a player:** Tap on a player in the list to select them.
3.  **Search for a song:** Use the search bar to find a song.
4.  **Select a song:** Tap on a song from the search results to assign it to the selected player.
5.  **Control playback:** Use the "Play," "Pause," and "Stop" buttons to control the music.
